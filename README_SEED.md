# 初期データの作成手順

## 重要: データベースのマイグレーションを先に実行してください

**エラーが発生する場合**: `profiles`テーブルが存在しないエラーが発生している場合は、先にデータベースのマイグレーションを実行する必要があります。

詳細は `SETUP_DATABASE.md` を参照してください。

## 1. ユーザーアカウントの作成

1. ブラウザで `http://localhost:3000/auth/signup` にアクセス
2. 以下の情報でアカウントを作成：
   - メールアドレス: `test@example.com`
   - パスワード: `test123456`
   - 農場名: `テスト農場`（任意）

## 2. ログイン

1. `http://localhost:3000/auth/login` にアクセス
2. 作成したアカウントでログイン：
   - メールアドレス: `test@example.com`
   - パスワード: `test123456`

## 3. 初期データの作成

1. ログイン後、`http://localhost:3000/admin/seed` にアクセス
2. 「初期データを作成」ボタンをクリック
3. 以下のデータが作成されます：
   - プロフィール（テスト農場）
   - 収益データ（5件）
   - 支出データ（6件）
   - 補助金・助成金データ（3件）
   - 畜群データ（3件）

## 4. データの確認

初期データ作成後、以下のページでデータを確認できます：
- ダッシュボード: `http://localhost:3000/dashboard`
- 収益: `http://localhost:3000/revenue`
- 支出: `http://localhost:3000/expenses`
- 畜群管理: `http://localhost:3000/herds`
- 補助金・助成金: `http://localhost:3000/subsidies`

## 注意事項

- 初期データは既存のデータと重複しないように作成されます
- 同じデータを複数回作成しても問題ありません（重複は無視されます）
- データは現在のユーザーに紐づいて作成されます
